## Использование приложения
1. Клиентское приложение:
   
- Нажмите "Обзор..." для выбора файла
- Программа автоматически вычислит хэш-сумму
- По умолчанию включено использование криптографической соли (требование 5.2)
- Нажмите "Отправить файл на сервер"

2.Серверное приложение:

- Проверяет целостность полученных файлов
- Сохраняет файлы в папку "ReceivedFiles"
- Отправляет результат проверки клиенту
